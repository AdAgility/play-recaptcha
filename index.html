play-recaptcha

Google reCAPTCHA integration for the Play Framework

Play reCAPTCHA Module

This is a Play Framework module, for Scala 2.10 and Play 2.3, to provide integration with Google reCAPTCHA in a reactive (non-blocking) manner.

See the play-recaptcha GitHub Project for more details and documentation.
